# Founder OS Vision v2

Status: Approved

## Platform Definition

Founder OS is an AI-powered product creation platform for designing, planning, building, validating, and launching high-quality applications and websites through guided workflows and automation.

## Product Relationship

Founder OS is the platform.

Natural Nation is Project #1 built and managed by Founder OS.

## Separate Products

- Founder OS platform
- Natural Nation branding website
- Natural Nation application with Duey

The three products are related but remain separate.

## Founder OS Responsibilities

- guided project setup
- requirements discovery
- wireframe creation and revision
- design approval
- branding and asset coordination
- implementation planning
- repository and workflow orchestration
- validation and approval gates
- deployment coordination

## Experience Rule

The Founder experience should remain simple and guided.

Technical command identifiers, payloads, repositories, workflows, and automation details should remain behind the interface unless needed for review or troubleshooting.

## First Working Model

The first operational model should complete one end-to-end workflow:

1. Create a project
2. Capture requirements
3. Generate a wireframe
4. Revise the wireframe
5. Lock the approved design
6. Generate a branding website
7. Generate an application foundation
8. Validate the outputs
9. Present the results for Founder approval

## Scope Rule

Reusable product-creation capabilities belong in Founder OS.

Natural Nation wellness logic, Duey behavior, protocols, and user experience belong in the Natural Nation project.
