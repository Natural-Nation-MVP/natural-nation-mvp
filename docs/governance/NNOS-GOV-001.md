# NNOS-GOV-001 GitHub Governance Sync Standard

Status: Approved

## Purpose

GitHub is the source of truth for Natural Nation and Founder OS governance.

ChatGPT supports planning, drafting, review, and implementation guidance.

Approved decisions should be recorded in GitHub so the project can be continued from repository documents.

## Operating Rule

Before major work, load the project state, architecture, readiness report, release roadmap, governance files, and decision records.

After approved work, update the related GitHub documents.

## Required Records

- Project state
- Architecture standards
- Readiness reports
- Release roadmaps
- Decision records
- Build packages
- AI handoff standards

## Founder Approval Rule

Founder approval is required before changing architecture, deployment, design standards, workspace layout, release roadmap, governance, or build package standards.
