# ADR-002 GitHub Project Intelligence

Status: Approved

## Decision

GitHub is the canonical project intelligence platform for Natural Nation and Founder OS.

## Context

Important project information had been spread across ChatGPT conversations, screenshots, approvals, and repository files.

## Standard

Approved knowledge should be stored in GitHub documentation, issues, decision records, release roadmaps, and build packages.

## Operating Rule

ChatGPT should load GitHub documentation before major work and update GitHub records after approved work.

## Goal

No critical Natural Nation or Founder OS decision should exist only inside a ChatGPT conversation.

## Founder Approval

Approved.
