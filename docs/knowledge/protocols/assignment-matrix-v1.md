# KB-PROTOCOL-003 — Protocol Assignment Matrix v1

Status: Founder Approved

## Purpose

The assignment matrix defines how Natural Nation selects daily wellness protocols from onboarding inputs and user state.

## Assignment Limits

- Daily Foundations is always assigned.
- Maximum active protocols: 3 total.
- Structure: 1 foundation plus up to 2 supporting protocols.

## Priority Hierarchy

1. Safety
2. Sleep
3. Recovery
4. Stress
5. Energy
6. Mental Clarity
7. Digestion
8. Weight
9. Movement
10. Hydration
11. Longevity

## Assignment Rules

Safety overrides every other priority.

Sleep, recovery, and stress concerns are handled before goal-based optimization.

Hydration may be assigned as a supporting protocol when hydration is poor and no higher-priority concern takes the available slot.

If two protocol candidates are equal, the higher item in the priority hierarchy wins.

## Deterministic Rule

Same inputs must produce the same output.
