## Purpose

The Protocol Library defines the approved wellness protocol categories used by Natural Nation.

## Active Protocols

1. Daily Foundations
2. Recovery Reset
3. Sleep Restoration
4. Energy Optimization
5. Stress Resilience
6. Mental Clarity
7. Weight Loss Foundation
8. Digestive Support
9. Hydration and Mineral
10. Movement and Mobility
11. Longevity Foundation

## Core Rules

Daily Foundations is always active.

A user may receive one foundation protocol plus up to two supporting protocols.

Safety concerns override all other assignments.

Goals help guide personalization, but goals do not override the wellness priority hierarchy.

The same inputs should produce the same protocol assignment.

## First Win Principle

Each protocol should help the user feel an early win within the first several days when possible.

## Recognition Principle

Duey should recognize consistency and progress at meaningful check-in points.
