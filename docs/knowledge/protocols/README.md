# KB-PROTOCOL-001 — Protocol System

Status: Founder Approved

## Purpose

The protocol system assigns simple daily wellness actions based on user needs, onboarding inputs, and Natural Nation wellness priorities.

## Approved Protocol Principles

- Daily Foundations is always active.
- Maximum active protocols: one foundation plus up to two supporting protocols.
- Safety concerns override all other assignments.
- Goals do not override wellness priority hierarchy.
- Same inputs must produce the same assignment.
