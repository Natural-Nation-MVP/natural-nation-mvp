# KB-AI-002 — AI Context Loading Standard

Status: Founder Approved

## Purpose

This standard defines the minimum project context AI agents should load before major Natural Nation or Founder OS work.

## Required Startup Context

Major work should begin with:

- docs/knowledge/INDEX.md
- docs/PROJECT_STATE.md
- docs/SESSION-LOG.md
- docs/releases/RELEASE-3-ROADMAP.md
- docs/governance/SYNC-STANDARD.md
- relevant knowledge domain records
- relevant ADRs or governance records

## Role Use

Art uses this context for architecture.

Codex uses this context for implementation.

Gemini uses this context for design review.

GPose uses this context for summaries, prompts, and planning.

Duey uses this context for wellness and mentor behavior.

## Completion Rule

After execution, affected knowledge records and project tracking must be updated.
