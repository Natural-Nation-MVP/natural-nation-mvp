# KB-AI-001 — AI Operations

Status: Founder Approved

## Purpose

AI Operations defines the roles, handoffs, execution order, build package routing, review ownership, and approval flow for Natural Nation and Founder OS.

## AI Roles

- Art: architecture, systems, standards, and ADRs
- Codex: implementation and technical delivery
- Gemini: design review and UX validation
- GPose: prompts, planning, summaries, and founder-facing documentation
- Duey: wellness knowledge and mentor behavior
- Founder: final approval for locked decisions

## Operating Rule

Major AI work should load the Knowledge Base index, project state, release roadmap, session log, and relevant governance before execution.
