# KB-FOUNDER-002 — Founder OS Architecture Reference

Status: Founder Approved

## Purpose

This document records the approved Founder OS architecture direction for Release 3.

## Canonical Runtime

Founder OS public runtime is:

`docs/founder-os/`

Legacy locations should not be treated as canonical runtime paths.

## Workspace Model

Founder OS is organized around operational workspaces:

- Mission Control
- Knowledge Graph
- Build Studio
- Repository Intelligence
- AI Operations
- Reports
- Settings

## Production Layout Standard

The approved layout uses:

- left sidebar navigation
- top workspace heading
- compact metrics strip
- main workspace panel
- right context rail
- bottom command bar
- iPad portrait and landscape support

## Build Studio Baseline

Build Studio is the validated production baseline. It includes selected build item, target selector, impact explanation, plain English summary, package preview, queue metadata, right rail utilities, and synchronized bottom bar.

## Source of Truth

GitHub is the source of truth for Founder OS state, documentation, roadmap, decisions, and knowledge records.
