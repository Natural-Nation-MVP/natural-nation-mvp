# KB-DESIGN-002 — Design System v1

Status: Founder Approved

## Purpose

This record summarizes the approved Natural Nation and Founder OS design direction.

## Natural Nation Design Direction

Natural Nation should feel premium, calm, natural, simple, and supportive. The visual system should use wellness-oriented spacing, soft cards, clear hierarchy, and friendly AI mentor presence.

## Founder OS Design Direction

Founder OS should feel like a professional operating workspace. It uses sidebar navigation, compact metric cards, glass-style panels, right rail context, bottom command bar, and iPad-friendly responsive layouts.

## Approved UI Standards

- Clear screen hierarchy
- Mobile-first behavior for Natural Nation
- iPad-friendly behavior for Founder OS
- Consistent card system
- Consistent score card behavior
- Clean spacing and readable type
- Support for multi-device mockups when generating screens
