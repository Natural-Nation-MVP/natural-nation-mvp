# KB-PRODUCT-003 — Onboarding

Status: Founder Approved

## Purpose

This record tracks the approved Natural Nation onboarding direction.

## Flow Areas

- Welcome
- Goal Selection
- Goal Confirmation
- Food Preferences
- Dietary Preferences
- About You
- Duey Summary
- Blueprint Creation
- Blueprint Reveal
- Day 1 Dashboard Entry

## Experience Goal

The first session should give the user a clear early win, a personal plan, and a simple first daily mission.

## Duey Behavior

Duey should confirm choices in plain language, explain why the plan fits, and keep the user supported.
